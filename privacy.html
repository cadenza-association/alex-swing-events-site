Politique de confidentialité

Dernière mise à jour : 30 mai 2026

1. Collecte des données

Alex Swing Events utilise une application de publication automatisée de contenus vidéo sur les réseaux sociaux (TikTok, Instagram, Facebook, YouTube). Cette application publie des vidéos de concerts et d'événements musicaux sur nos comptes officiels.

L'application ne collecte aucune donnée personnelle d'utilisateurs tiers. Elle accède uniquement aux comptes de réseaux sociaux appartenant à Alex Swing Events.

2. Utilisation des données

Les données traitées par notre application sont exclusivement :

  • Les fichiers vidéo des concerts et événements Alex Swing Events
  • Les métadonnées associées (titre, description, hashtags)
  • Les tokens d'accès aux API des réseaux sociaux (TikTok, Instagram, Facebook, YouTube)

Ces données sont utilisées uniquement pour publier du contenu sur nos propres comptes de réseaux sociaux.

3. Stockage des données

Les vidéos sont temporairement hébergées sur Cloudinary (service de stockage cloud) le temps de la publication, puis peuvent être supprimées. Les tokens d'accès sont stockés de manière sécurisée sur notre infrastructure.

4. Partage des données

Aucune donnée personnelle n'est partagée avec des tiers. L'application interagit uniquement avec les API officielles des plateformes de réseaux sociaux.

5. Contact

Pour toute question relative à la confidentialité : alex.swing.events@gmail.com
